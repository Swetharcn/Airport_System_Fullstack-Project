# AirAssist – Smart Airport System

AirAssist is a smart airport assistance platform developed using the MERN stack to improve passenger convenience and airport management efficiency. The system provides features such as AI chatbot assistance, indoor navigation, OTP authentication, crowd monitoring, baggage tracking, and airport service discovery.

## Features

* AI-powered chatbot using Groq and Gemini APIs
* Smart indoor navigation using Dijkstra’s Algorithm
* OTP-based authentication using Fast2SMS
* Crowd density monitoring and visualization
* Flight and baggage tracking
* Boarding reminders and notifications
* Airport services module

## Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Axios

Backend:

* Node.js
* Express.js
* MongoDB

APIs:

* Groq API
* Gemini API
* Fast2SMS API

## Installation

Clone the repository:
git clone <repository-link>

Install dependencies:
npm install

Run the project:
npm run dev
